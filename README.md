# Social Media Database Project

## Overview
A SQL-based database design for a social media platform with users, posts, comments, likes, and followers.

## Features
- Normalized SQL schema
- ER Diagram (exported from MySQL Workbench)
- Triggers, Views, Stored Procedures
- Example queries (Top posts, Leaderboard, User feed)

## Files
- social-media Database.sql → Schema + queries
- ER_diagram.png → Database ER diagram
- Social Media Database Report.pdf → 2-page final report

## How to Run
1. Import social_media.sql into MySQL Workbench
2. Run queries and explore views/triggers
3. Use data to test outputs
